# Hotel Booking Web Application

## Overview

This project is a full-stack hotel booking application developed using React, Node.js, Express.js and MySQL as part of a collaborative university software development project.

The application allows users to:

- Browse available hotel rooms
- Search by room name or location
- View room details
- Register and log in
- Book hotel rooms
- Submit room ratings
- View bookings on a personalised dashboard

## Running the Project

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
npm install
node server.js
```
